# World Bank Economic Indicators Pipeline

ETL pipeline analyzing global economic data (2010-2025) using PySpark, BigQuery, and Looker Studio.

## Files
- `MgmtAccessPreprocessing.ipynb` - Data cleaning with PySpark
- `Queries.sql` - Economic analysis queries
- `Report.docx` - Project documentation

## Tools
- **Processing**: PySpark
- **Storage**: Google Cloud Storage + BigQuery
- **Visualization**: BigQuery + Looker Studio
